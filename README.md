## 🚀 Funcionalidades

- CRUD completo para gerenciamento de produtos
- Integração com banco de dados H2 (in-memory) para desenvolvimento rápido e testes
- Preparado para uso com bancos MySQL/PostgreSQL (configuração fácil)
- Documentação automática da API com Swagger/OpenAPI
- Estrutura de projeto limpa e organizada em camadas (Controller, Service, Repository)
- Configurações para desenvolvimento local e ambiente de produção

---

## 🛠️ Tecnologias Utilizadas

- Java 21
- Spring Boot 3.3.11
- Spring Data JPA
- H2 Database (desenvolvimento)
- Swagger/OpenAPI (documentação)
- Maven (gerenciador de dependências)
